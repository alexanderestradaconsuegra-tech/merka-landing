❓ Preguntas frecuentes

Todo lo que
necesitas saber.

¿Necesito instalar algo? +
No. Merka POS funciona directamente en el navegador. Solo abres la URL desde cualquier dispositivo. Opcionalmente puedes "instalarlo" en la pantalla de inicio de tu celular para que se vea como una app nativa.
¿Funciona sin internet? +
Sí. El sistema tiene una capa de caché que permite seguir vendiendo aunque se corte el internet. Cuando vuelve la conexión, los datos se sincronizan automáticamente con la base de datos.
¿Cuántos dispositivos puedo usar? +
Los datos se sincronizan en tiempo real entre todos los dispositivos. Puedes tener el sistema abierto simultáneamente en la caja, un tablet de inventario y el celular del dueño.
¿Funciona con balanzas? +
Sí. Compatible con balanzas electrónicas que generan código EAN-13 con peso embebido (formato estándar). Al pistoliar el código, el sistema detecta el producto y el peso automáticamente — no necesitas ingresar nada manualmente.
¿Puedo importar mi inventario actual? +
Sí. Merka POS acepta importación desde Excel (.xlsx). Exporta tu inventario actual a Excel con las columnas indicadas y lo cargamos en segundos.
¿Puedo cancelar en cualquier momento? +
Sí. Sin contratos ni permanencia mínima. Cancelas cuando quieras desde tu panel. No hay cargos adicionales ni penalidades.
